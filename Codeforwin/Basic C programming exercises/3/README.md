Write a C program to input two numbers and perform all arithmetic operations.

Problem link: https://codeforwin.org/2015/05/c-program-to-perform-all-arithmetic-operations.html

## Example
## Input:
First number: 10

Second number: 5
## Output:

Sum = 15
Difference = 5

Product = 50

Quotient = 2

Modulus = 0
